# 💰 Finance Dashboard (React + Recharts)

A modern **Finance Tracking Dashboard** built using **React.js** and **Recharts**.
This application helps users track transactions, visualize spending, and gain smart financial insights.

---

## 🚀 Features

✅ Add & manage transactions (Income / Expense)
✅ 📊 Interactive charts (Pie & Line) using Recharts
✅ 💡 Smart Insights (highest spending category)
✅ 🔍 Search & filter transactions
✅ 📱 Fully responsive modern UI
✅ 🎨 Glassmorphism dashboard design
✅ ⚡ Smooth animations and transitions

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 📊 Recharts (for charts & analytics)
* 🎨 CSS3 (Custom modern UI design)
* 📦 JavaScript (ES6+)

## 📦 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/finance-dashboard.git
cd finance-dashboard
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the App

```bash
npm start
```
